# 🛠 Requisição de Materiais

Sistema web desenvolvido para automatizar a solicitação de materiais operacionais e reposição de itens com defeito nos postos de serviço.

A aplicação permite que supervisores solicitem materiais de forma rápida e organizada, registrando informações importantes para controle interno e acompanhamento das demandas.

Semelhante ao sistema de uniformes, este projeto foi adaptado para atender às necessidades de reposição e manutenção operacional.

## 🚀 Funcionalidades

✅ Solicitação de materiais operacionais  
✅ Registro de itens com defeito ou avarias  
✅ Preenchimento de informações do posto de serviço  
✅ Envio automático para controle interno  
✅ Organização das solicitações em banco de dados  
✅ Interface responsiva para desktop e mobile  
✅ Acompanhamento das demandas operacionais  

---

## 🛠 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Google Sheets (armazenamento de dados)  
- Google Apps Script (automação e integração)  

---

## 🎯 Objetivo do Projeto

O objetivo deste sistema é centralizar e automatizar o processo de solicitação de materiais e reposição de itens com defeito.

Antes da automação, o processo era manual e sujeito a falhas.

Com a implementação:

✔️ Maior controle das solicitações  
✔️ Melhor organização das demandas  
✔️ Redução de erros manuais  
✔️ Agilidade no atendimento aos postos de serviço  
✔️ Melhor rastreabilidade dos pedidos
